<h2> Interim Submission</h2> 
For my interim submission of the Closet Organiser App, I have implemented the following core features:

1. Lab Work & Exercises
Completed all lab work and exercises as part of the course curriculum to enhance understanding of Android development and apply practical skills.

2. CRUD Operations
  Create, Read, Update, and Delete functionality for managing clothing items.

  Create: Users can add new clothing items to their closet.

  Read: Users can view the details of their clothing items.

  Update: Users can edit the information for existing clothing items.

  Delete: Users can remove clothing items they no longer want.

3. Navigation & Menus
  Implemented navigation menus to allow users to easily return to the Home screen from other sections of the app.

4. Design
  The app has a blue color scheme for a clean and modern look.

  Included icons for search and favorites (heart icon) to enhance user interaction.

5. UI Elements
  Added various UI elements to enhance usability, such as:

  Date Picker: For users to select the last worn date of an item.

  Number Input: For sizing.

  Multi-line Text Fields: For detailed descriptions or comments about clothing items.

6. UI Components
  Used dropdown lists for selecting clothing seasons (e.g., Summer, Winter).

  Date Picker for tracking when an item was last worn.
